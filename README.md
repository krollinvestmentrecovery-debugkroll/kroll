# KROLL - Professional Financial Recovery Services

## 📋 Overview

KROLL is a comprehensive web-based platform designed for professional financial recovery services. The platform provides a secure, password-protected environment where clients can track their investments, manage withdrawals, and view real-time portfolio performance.

## 🔐 Features

### Website Lock System
- **Full-page password protection** - Visitors cannot access any content without the correct password
- **Glassmorphism design** - Modern, professional dark theme with blur effects
- **Show/Hide password toggle** - Convenient password visibility control
- **Loading animation** - Visual feedback during password verification
- **Session management** - Stay authenticated for 24 hours
- **Lock/Unlock functionality** - Users can lock the website with one click
- **Back navigation prevention** - Cannot bypass lock using browser back/forward buttons

### Core Features
- **Live Investment Tracker** - Real-time chart showing portfolio performance
- **Dashboard** - View investment summary and withdrawal details
- **Manual Profit Entry** - Type your own profit amount (R 1,805,100 default)
- **PDF Report Generation** - Download detailed portfolio reports
- **Banking Details Form** - Enter receiving account information
- **Take Profit Progress** - Visual progress bar showing TP status (0-99%)
- **User Authentication** - Login/Register system with local storage

### Security
- Password-protected website lock screen
- User authentication with email/password
- Session timeout after 24 hours
- Protected HTML pages, dashboards, PDFs, and downloads
